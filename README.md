### Overview

**One More Roll** is a frontend web application designed to provide a seamless and personalized online shopping experience. Users can browse products, manage their purchases, and customize their interface with options like dark mode and language selection (English and Korean). Built with React, Firebase, and Cloudinary, this project showcases a scalable approach to web development.

**One More Roll**은 사용자에게 매끄럽고 맞춤형의 온라인 쇼핑 경험을 제공하는 프론트엔드 웹 애플리케이션입니다. 사용자는 다양한 제품을 탐색하고, 구매를 관리하며, 다크 모드와 영어 및 한국어 선택과 같은 기능으로 인터페이스를 자신에게 맞게 조정할 수 있습니다. React, Firebase, Cloudinary로 구축된 이 프로젝트는 확장 가능한 웹 개발 접근 방식을 보여줍니다.

---

### Features

-   **User Authentication**: Secure user login and registration powered by Firebase Authentication.   
    **사용자 인증**: Firebase 인증을 통해 안전한 사용자 로그인 및 등록 기능을 제공합니다.

-   **Conditional Menu and Feature Access**: Depending on the account you log in with, some menus and features are activated or deactivated to provide a personalized experience.  
    **조건부 메뉴 및 기능 접근**: 로그인하는 계정에 따라 일부 메뉴와 기능이 활성화되거나 비활성화되어 맞춤형 경험을 제공합니다.

-   **Product Management**: Browse a variety of products and add items to your wishlist or shopping cart.   
    **제품 관리**: 다양한 제품을 탐색하고, 원하는 제품을 위시리스트나 장바구니에 추가할 수 있습니다.

-   **Dark Mode**: Switch between light and dark themes for a comfortable browsing experience.   
    **다크 모드**: 라이트 모드와 다크 모드 전환을 통해 편안한 브라우징 경험을 제공합니다.

-   **Language Selection**: Switch between English and Korean for a localized experience.   
    **언어 선택**: 영어와 한국어를 전환을 통해 지역에 맞춤화된 경험을 제공합니다.

-   **Responsive Design**: Fully responsive layout ensuring a great experience on both desktop and mobile devices.   
    **반응형 디자인**: 데스크탑과 모바일 장치 모두에서 훌륭한 사용자 경험을 보장하는 완전한 반응형 레이아웃을 제공합니다.
